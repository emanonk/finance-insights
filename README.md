## 🚀 Overview

Finance Insights is a self-hosted application that helps users upload
bank statements, parse transactions, categorize spending, and generate
insights.

## ✨ Features

-   Multi-bank statement upload
-   Parser detection & normalization
-   Merchant identification & tagging
-   Reports & analytics
-   Manual corrections & duplicate detection

## 🏗️ Architecture

Frontend (React) → Backend (Go API) → PostgreSQL + File Storage

## 🧰 Tech Stack

-   Go 1.25
-   React + TypeScript + Tailwind
-   PostgreSQL
-   Docker Compose
